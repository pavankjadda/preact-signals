# React project that demonstrates Preact signals

React App skeleton project that can be used as baseline/skeleton for new React projects

1. React 18.x
2. Preact Signals
3. React Router 7.x
4. Material UI 5.x
5. Axios
6. React Hook Form
7. Vite 3.x

### Run the Project on Local Machine

1. Clone the project
2. Start react project using PNPM command
    ```shell
    pnpm dev
    ```
3. Go to http://localhost:3000 to access the website
